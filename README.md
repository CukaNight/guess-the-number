# Угадай число

Консольная игра на Python, где компьютер загадывает случайное число, а игрок пытается его угадать.

## Версия

**v1.0** — базовая версия

---

## Правила

- Компьютер загадывает число в заданном диапазоне
- Игрок вводит свой вариант
- Программа подсказывает: «больше» или «меньше»
- Игра продолжается до угадывания
- За каждую победу начисляются очки

---

## Как запустить

```bash
python guess_the_number.py
