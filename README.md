# Угадай число

Консольная игра на Python. Компьютер загадывает число, игрок угадывает.

## Версия

**v1.0** — полная версия с выбором сложности и системой очков

---

## Возможности

- 4 уровня сложности (лёгкий, средний, сложный, невозможный)
- Ограничение по попыткам для каждого уровня
- Начисление очков за победу (чем сложнее — тем больше очков)
- Подсказки «больше» или «меньше»
- Счётчик очков на весь сеанс игры

---

## Как запустить

```bash
python guess_the_number.py
